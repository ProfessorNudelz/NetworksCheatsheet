Check if Equipment is not 'clean':

      Switch# show startup-config
      Switch# show flash:
      Switch# show vlan brief

To Clean Equipment:

      Switch# write erase
      Switch# delete vlan.dat
      Switch# reload

3 Essential Commands:

      Switch(config)# no ip domain lookup
      Switch(config)# line console 0
      Switch(config-line)# logging synchronous
      Switch(config-line)# exec-timetable 0 0

Global Config:

      Switch(config)# hostname hostname
      Switch(config)# no hostname
      Switch(config)# banner motd #string#

Config Files:

      Switch# show startup-config
      Switch# show running-config
      Save running-config:
      Switch# copy running-config startup-config
      Restore to startup-config:
      Switch# reload

Verify Connectivity:

      Switch# show ip interface brief

Disable Ethernet Interfaces:

      Select a single interface:
      Switch(config)# interface gigabitEthernet 1/0/1
      Select a range of interfaces:
      Switch(config)# interface range gigabitEthernet 1/0/1 - 14
      Turn off interface:
      Switch(config-if)# shutdown
      Turn on interface:
      Switch(config-if)# no shutdown

VLANs:

      Create a VLAN:
      Switch(config)# vlan number
      Switch(config-vlan)# name name
      Switch(config-vlan)# exit

      Configure membership of interfaces to the VLAN:
      Switch(config)# interface range gigabitEthernet 1/0/1 - 14
      Switch(config-if)# description Users VLAN
      Switch(config-if)# switchport mode access
      Switch(config-if)# switchport access vlan 10
      Switch(config-if)# no shutdown
      To remove an interface from a vlan:
      Switch(config-if)# no switchport access vlan

      Delete a VLAN:
      Switch(config)# no vlan 10

      Show VLAN information:
      Switch# show vlan brief

Change Management VLAN:

      Switch(config)# interface vlan 1
      Switch(config-vlan)# shutdown
      Switch(config-vlan)# exit
      Switch(config)# vlan 99
      Switch(config-vlan)# name Management
      Switch(config-vlan)# exit
      Switch(config)# interface vlan 99
      Switch(config-vlan)# description Management
      Switch(config-vlan)# ip address 192.168.99.3 255.255.255.0
      Switch(config-vlan)# no shutdown
      Switch(config-vlan)# exit

VLAN Trunking:

      Switch(config)# interface gigabitEthernet id
      Switch(config-if)# switchport mode trunk

      Change native VLAN:
      Switch(config-if)# switchport trunk native vlan 10

      Specify list of VLANs allowed to be on trunk list:
      Switch(config-if)# switchport trunk allowed vlan 10,20,30
      Switch(config-if)# exit

      Show trunk ports:
      Switch# show interfaces trunk

      When using 3650 switches in Packet Tracer or ATC330, use:
      Switch(config-if)# switchport trunk encapsulation dot1q
      BEFORE:
      Switch(config-if)# switchport mode trunk

Switchport Port-Security:

Configure Port Security

      Select interface:
      Switch(config)# interface gigabitEthernet id

      Enable Port-Security (interface must be in access mode first):
      Switch(config-if)# switchport port-security

      Configure maximum allowed MAC Addresses:
      Switch(config-if)# switchport port-security maximum N

      Shutdown port upon breach:
      Switch(config-if)# switchport port-security violation shutdown
      To re-enable port, shutdown and then no shutdown port

      Drop packets from unknown source:
      Switch(config-if)# switchport port-security violation restrict

      Sticky MAC Addresses (can't be changed):
      Switch(config-if)# switchport port-security mac-address sticky
      Switch(config-if)# no switchport port-security mac-address sticky

      Allocate a specific MAC Address:
      Switch(config-if)# switchport port-security mac-address XXXX.XXXX.XXXX.XXXX

Explore Port Security

      Switch# show mac address-table
      Switch# show port-security
      Switch# show port-security interface gigabitEthernet 1/0/1
      Switch# show port-security address
      Switch# show interface gigabitEthernet 1/0/1

SSH

      Verify SSH support:
      Switch# show ip ssh
      
      Configure domain name:
      Switch(config)# ip domain-name cisco.com
      
      Generate RSA key-pairs:
      Switch(config)#cyrpto key generate rsa 
      
      Configure user authentication:
      Switch(config)# username admin secret ccna

      Configure vty lines:
      For Switch:
      Switch(config)# line vty 0 15
      For Router:
      Router(config)# line vty 0 4
      
      Switch(config-line)# transport input ssh
      Switch(config-line)# login local
      Switch(config-line)# exit
      
      Enable SSH version 2:
      Switch(config)# ip ssh version 2

Configure Default Gateway

      Switch(config)# ip default-gateway 192.168.0.1

Loopback Interface (Router)

      Router(config)# interface lo0
      Router(config-if)# description Loopback test interface
      Router(config-if)# ip address 209.165.200.225 255.255.255.224
      Router(config-if)# no shutdown
      Router(config-if)# exit

Router on a Stick

      Enable the Physical Interface:
      Router(config)# interface g0/0/1
      Router(config-if)# no shutdown
      Router(config-if)# exit

      Create the sub-interface:
      Router(config)# interface g0/0/1.99
      Router(config-subif)# description Connection to management VLAN
      Router(config-subif)# encapsulation dot1Q 99
      Router(config-subif)# ip address 192.168.1.1 255.255.255.0
      Router(config-subif)# no shutdown
      Router(config-subif)# exit

      To show routing table:
      Router# show ip route

      Ports on switch to router must be in trunk mode

Spanning Tree

      Show current spanning tree configuration:
      Switch# show spanning-tree

      Change Port Cost:
      Switch(config)# interface g1/0/1
      Switch(config-if)# spanning-tree cost 18
      To undo port cost change:
      Switch(config-if)# no spanning-tree cost 18
